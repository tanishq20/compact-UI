Image

Images that work well on devices with widely differing screen sizes and resolutions.

Responsive Image

To obtain this result, use img tag inside a div tag with img-responsive class name.

Rounded Image

To obtain this result, use img tag with image and img-rounded class name.

Square Image

To obtain this result, use img tag with image class name. (Note: By default image size is fixed at 150x150px.)
