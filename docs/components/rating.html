Rating

Ratings provide insight regarding others' opinions and experiences and can allow the user to submit a rating of their own.

Rating Icon

To obtain this result, use div tag with rating class name. Use rating-icon and rating-checked class to manipulate icon. Note: If icon is rating-checked then replace far with fas class name.

Rating Badge

To obtain this result, use div tag with rating-badge class name. Use rating-badge-text class for text and rating-badge-icon class for icon.
