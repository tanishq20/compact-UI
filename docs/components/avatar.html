Avatar

Avatar component is an image that represents an object, such as an account or user.

Images Avatar

To obtain this result, use img tag with avatar class name.

Size Avatars

To obtain this result, use img tag with avatar and sm-avatar | md-avatar | lg-avatar class name.

Square Avatars

To obtain this result, use img tag with avatar, square-avatar and sm-avatar | md-avatar | lg-avatar class name.
